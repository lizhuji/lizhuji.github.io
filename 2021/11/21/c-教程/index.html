各类基础语法

if else : 你要钱还是要命

当然要钱喽！
但这句话确实很适合形容if和else，一般讲，我们称其为判断语句
解释一下，if顾名思义，如果的意思，那么 else 就是：如果不是，那么。。。。
用代码解释就是：

$ if("你要钱"){
         那么你就没命
   } else {
         那么你就没钱
   }

很显然，我们看出来了，使用if…..else…..语句来让没脑子的电脑给我们判断一下，就挺。。。。。哈哈哈？
（当然了，人家没脑子也算的比你快✧(≖ ◡ ≖✿ ）

言归正传，我们用if…else…让电脑判断一些东西，当电脑判断完时，电脑会告诉你是对的还是错的，我们把对的叫做 ： True，错的叫做 False

$ if (1 + 1 == 2) { //让电脑判断
        cout << "Yes" << endl;//如果返回的是正确，那么输出这段代码
 } else {
        cout << "No" << endl;//如果返回的是错误，那么输出这段代码
 }

好了！那么有人会问了，那这else是什么鬼东西？
好问题。。。
你就把它当做和if相反的东西啦！
就比如说 

$ if ( 1+1 == 2)

那么相反不就是

$ if ( 1+1 != 2)

(ps:不是吧，不会有人不知道 “!=”是不等于的意思吧？）
咳咳，这个相反的语句呢，因为我们太懒了，就拿个else代替了
所以 else 就等于 if(1 + 1 != 2)
实在不理解，就这样说：
如果（1+1 == 2）
那么是对的
否则
是错的

再不理解就只能骂我了。。。。。。

OK，布置个回家作业：

$ #include <iostream>
   using namespace std;
   int main(){
        //写入判断语句代码，如果 1+1=12，那么输出 “耶~”，否则输出 “对了”
        return 0;
   }
